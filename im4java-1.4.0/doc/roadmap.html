Roadmap

Future Plans

The following releases will feature:

  • update interface definitions (catch up with new options)
  • do some benchmarking to support a better decision between JMagick and im4java
  • integration with java buildtools (e.g. ant and maven)
  • support of additional commandline tools (suggestions welcome)
  • add example programs
  • use JMagick/GMagick-JNI as backend (if feaseable)
